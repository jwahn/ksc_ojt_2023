# check_m+에서 MY에 관심종목 검색해서 등록하는 방법

1. 로그인 후 **마켓탭**으로 이동
 ![alt text](./check_m1.jpg)


2. **My 종목**에서 **+종목추가** 터치


3. 검색창에 원하는 종목명 또는 종목코드 입력
 ![alt text](./check_m2.jpg)


4. 검색결과가 나오면 **별표시** 터치
![alt text](./check_m3.jpg)


5. 다시 마켓탭으로 돌아가서 **My 종목**에 원하는 종목이 들어갔는지 확인
![alt text](./check_m4.jpg)
